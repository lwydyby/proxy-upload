== Building

To launch your tests:
```
./gradlew clean test
```

To package your application:
```
./gradlew clean assemble
```

To run your application:
```
./gradlew clean run
```

== Run with docker

1. To run docker-compose:
```
docker-compose up -d
```

The service port is 9000



== environment:

host: The host to route to
